# cpp00
L’objectif de cpp00 est de découvrir les bases de la programmation orientée objet en C++, en partant des notions déjà connues en C.
Le projet introduit les concepts fondamentaux comme les classes, les attributs, les méthodes, l’encapsulation, les constructeurs et destructeurs, ainsi que l’utilisation de iostream à la place de printf.
Tout le projet est réalisé en C++98, sans utiliser les fonctionnalités modernes du C++.
Il s’agit de comprendre la logique du C++ “classique” et la transition du C vers le C++.
